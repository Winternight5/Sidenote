What is Sidenote?

The applicationâ€™s main features are rather simple, but effective. Broadly speaking, the application enables users to simply append tasks onto their to-do list, which is the main functionality of the web application. The application also allows users to make accounts, and persists their to-do lists upon login. We implemented special features such as being able to type and manually draw/write notes, sharing notes to other users via email (and allowing edit permissions), and real time editing of notes/todo lists between creator's and collaborators.

This project lies on a Python-Flask server side program that retrieves data from a PostgreSQL database as a backend to which keeps track of user data. We used standard tools such as Git as a version control and Travis CI for continuous integration. The feature of real time editing of notes/todo lists is possible via the integration of websockets. The technologies we used were: Python, Flask, Javascript, AJAX, HTML, CSS/SCSS, Materialize.css, Travis CI, and Heroku.
